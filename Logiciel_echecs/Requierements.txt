#Pour le projet de developpement du logiciel de tournoi d'échecs: 

# pas de dépendances externes nécessaires pour la version de base
# => les modules dont nous avons utilisé, sont intégrés dans Python par défaut (datetime, random,etc.), 
# donc nous n'avons pas eu besoin de les installer. Ils sont juste utilisés directement dans le code: 
# datetime:gérer les dates (ex. début/fin du tournoi);
# random:mélanger les joueurs de manière aléatoire;
# json:sauvegarder les données dans des fichiers .json;
# pathlib:  manipuler les chemins de fichiers facilement;
# typing: donner des indications de types (pour rendre le code plus lisible).

# modules de vérification du code à la fois qualitatif et le style
# => flake8 : outil principal d'analyse de code (linting ou pelluchage en français)
# Il repère les erreurs de style : ligne trop longues, espaces manquants, variables non utiisées, noms de fichiers incohérents. 

# => flake8-html, c'est une extension de flake8
# Il permet de générer un rapport visuel au format html; 
# Ce dernier montre toutes les erreurs flake8 --format=html --htmldir=flake8_rapport; 
# et se trouve dans le dossier "flake8_rapport/index.html".
